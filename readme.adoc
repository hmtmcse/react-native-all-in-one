
= React Native Expo Example Application

In this Application Listing all the useful component.


== Run Application
```bash
cd react-native-all-in-one
pnpm install
pnpm start

# For Android Device Run
pnpm expo prebuild --clean
pnpm expo run:android

# For Production Build
pnpm android:prod  # Custom command added in package.json

cd android
gradlew assembleRelease
adb install -r app/build/outputs/apk/release/app-release.apk
```

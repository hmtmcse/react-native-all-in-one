
== react-native-gesture-handler
```bash
pnpm add react-native-gesture-handler
npx expo install react-native-gesture-handler
```

`react-native-gesture-handler` makes touch interactions in React Native smooth, fast, and reliable.

== Install @react-native-reusables
```bash
pnpm add nativewind react-native-reanimated react-native-safe-area-context
pnpm add --save-dev tailwindcss@^3.4.17 prettier-plugin-tailwindcss@^0.5.11 babel-preset-expo
pnpm add lucide-react-native
pnpm dlx expo install react-native-svg

pnpm add -D @react-native-reusables/cli

# Check Files
* App.tsx
* components.json
* global.css
* tailwind.config.js

pnpm start --clear
pnpm dlx @react-native-reusables/cli@latest add --all
npx @react-native-reusables/cli doctor
```
